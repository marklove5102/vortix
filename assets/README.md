# Assets Directory

This directory contains media assets for the project:

- `dashboard.png` - Screenshot of the main dashboard (add this before publishing)

## Creating Screenshots

To create a screenshot for the README:

1. Run `sudo vortix` in a clean terminal
2. Take a screenshot (Cmd+Shift+4 on macOS)
3. Save as `dashboard.png` in this directory
4. Recommended size: 1600x900 or similar 16:9 ratio
